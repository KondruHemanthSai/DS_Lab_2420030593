# Diabetes Prediction Project

This project uses a Random Forest Classifier to predict diabetes.

## Model Performance

Our Random Forest model achieved the following metrics:

* **Cross-validated AUC:** 0.8388

### Test Set Metrics
* **Accuracy Score:** 0.7338
* **F1 Score:** 0.6822
* **Recall Score:** 0.8148
* **Precision Score:** 0.5867
